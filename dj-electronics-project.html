About the Project

DJ Electronics is a full-featured e-commerce platform specifically designed for DJ equipment sales. The application features user authentication (local and Google OAuth), product browsing by category and brand, detailed product pages with specifications and user reviews.

The platform includes shopping cart management, secure Stripe payment integration, and comprehensive testing with Jest, Vitest, and Cypress. Products are organized into categories including DJ Controllers, Booths, Lights, and Speakers, with detailed specifications and user review systems.

Key Features

  • Google OAuth Authentication
  • Product Filtering & Search
  • Stripe Payment Integration
  • Review System
  • Cart Management
  • Comprehensive Testing Suite

Technical Implementation

The application uses a React frontend with Node.js/Express backend, PostgreSQL database for data persistence, and Bootstrap for responsive design. Authentication is handled through Passport.js with both local and Google OAuth strategies. Payment processing is secured through Stripe integration.

The project includes extensive testing coverage with Jest for unit testing, Vitest for component testing, and Cypress for end-to-end testing, ensuring reliability and maintainability.

Technologies Used
React Node.js Express PostgreSQL Bootstrap Stripe Passport.js Jest Vitest Cypress
