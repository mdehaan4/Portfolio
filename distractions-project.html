About the Project

The Distractions Band Management System is a web application designed to streamline the entire process of managing live music events - from initial booking to final performance. The platform serves three main user types: Clients (book events), Musicians (receive gig offers, manage invoices), and Administrators (manage everything).

Features include automated invoice generation, calendar integration, instrument-based matching, push/email/SMS notifications, and comprehensive reporting. The system handles the complete workflow from client inquiries to musician payments, making band management efficient and professional.

Key Features

  • Role-based Access Control
  • Automated Notifications (Email/SMS/Push)
  • PayPal Payment Integration
  • Real-time Gig Management
  • Invoice Generation & Tracking
  • Calendar Integration (iCal)

About The Distractions Band Ltd

I am part of many functions bands as a Sound Engineer, but there aren't any that are quite like The Distractions Band. The band consists of over 60 musicians and is managed by one person. This means that efficiency and organisation is key.

A while ago, the Tech Lead/Drummer asked me if I could help build a system where clients and musicians could sign in and manage their respective events. I loved the sound of this project from the start and jumped at the opportunity. tRPC, Prisma, Next.js and NextAuth were used as I had been learning about these technologies at the time and they seemed like a good fit. The system is complete with password-less authentication, role based access control, admin console and file uploading. The design is mobile first, but fully responsive. React Admin is used to bootstrap the admin console and uploadThing to handle file uploads.
