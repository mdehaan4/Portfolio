About the Project

Jammin is a modern web application that integrates with the Spotify API to provide users with an enhanced playlist creation experience. Built with React, TypeScript, and Vite, the app implements Spotify's PKCE authentication flow for secure access.

Users can search for tracks, build custom playlists with an intuitive interface, and save their creations directly to their Spotify account. The application includes comprehensive end-to-end testing with Cypress to ensure reliability and user experience quality.

Key Features

  • Spotify PKCE Authentication
  • Music Search & Discovery
  • Custom Playlist Creation
  • Save to Spotify Account
  • Responsive UI
  • End-to-End Testing

Technical Implementation

The application leverages modern web technologies with React for the component architecture, TypeScript for type safety, and Vite for fast development and building. The Spotify Web API integration uses the secure PKCE (Proof Key for Code Exchange) authentication flow, ensuring user data protection.

CSS Modules provide scoped styling for maintainable code, while Cypress testing ensures the application works reliably across different scenarios and user interactions. The responsive design ensures optimal experience across desktop and mobile devices.

User Experience

The application provides an intuitive interface for music discovery and playlist management. Users can easily search for songs, preview tracks, and organize them into custom playlists. The seamless integration with Spotify means playlists are immediately available in the user's Spotify account across all their devices.
